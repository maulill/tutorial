1. sebelumnya saya telah membuat project laravel 12 breeze, sekarang tambahkan kolom role (admin, guru, dan siswa dengan default role guru), gender, adress, birth place, birth date, nomor hp, photo di tabel users
2. buatkan database tabel mapel(id, nama, durasi(jam)), pengampu(id, id mapel, id user dengan role guru) dan nilai (id, id siswa, id mapel, nilai)
3. saat user dengan role admin login, akan diarahkan ke dashboard admin
4. saat user dengan role guru login, akan diarahkan ke dashboard guru
5. saat user dengan role siswa login, akan stuck di halaman login dan beri peringatan bahwa user tersebut tidak dapat mengakses halaman
6. dasboard admin terdapat 3 crud, yaitu user, mapel dan pengampu(yang akan tampil di navigation yaitu kelola user(saat klik akan diarahkan ke halman kelola user yang menampilan crud user), kelola mapel (saat klik akan diarahkan ke halman kelola mapel yang menampilan crud mapel) dan kelola pengampu (saat klik akan diarahkan ke halman kelola pengampu yang menampilan crud pengampu))
7. crud user : terdapat button add user, dan ketika di klik, akan muncul form modal yang berisikan nama, email, pass, role, gender, adress, birth place, birth date, no hp. dibawah button add user terdapat tabel daftar user yang terdiri dari kolom kolom nomor, foto(bulat), nama, email, role, dan aksi yang terdapat 3 button detail(saat klik akan muncul modal dengan photo profile di tengah atas, dan dibawahnya lengkap detail user dengan contoh tampilan (nama:...)), edit(saat klik akan muncul modal yang berisikan textbox ynag didalam textbox pojok kiri atas bertuliskan nama dll dengan warna sedikit abu, dan isi dari detail itu berwarna hitam solid), delete (saat klik akan muncul modal untuk memastikan admin apakah yakin akan mengapus user ini)
8. crud mapel: terdapat buttom add mapel, dan ketika di klik akan muncul modal ynag berisikan nama dan durasi(jam), dan terdapat tabel daftar mapel di bawah button add mapel, yang berisikan kolom nomor, nama mapel, durasi(jam), dan aksi yang terdiri dari edit dan delete, dengan semua posisi isi kolom tabel berada di tengah
9. crud pengampu: terdapat button add pengampu, saat di klik akan muncul modal yang berisikan textbox dropdown mapel yang sudah terdaftar, dan textbox dropdown user dengan id guru) dan dibawah button add pengampu terdapat tabel daftar pengampu denagan kolom nomor, nama mapel, guru
10. dashboard guru menampilan tabel daftar mapel yang diampu, dengan kolom nomor, mapel, aksi nilai (saat klik button nilai di kolom aksi, akan muncul modal yang berisikan nama mapel read only, textbox dropdown daftar user dengan role siswa(akan otomatis update jika ada siswa yang nilainya sudah tersimpan), dan input nilai) dan detail yang jika di klik akan mengarahkan ke halaman yang berisikan tabel daftar nilai yang berisikan kolom nomor, nama siswa, nilai dan aksi yang terdapat button edit(saat diklik akan muncul modal nama siswa tsb read only dan nilai) dan delete
